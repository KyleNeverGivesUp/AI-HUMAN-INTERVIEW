## 🎯 完整操作流程

### 操作顺序：

1. ✅ 打开 `http://localhost:3000/digital-human`
2. ✅ 点击 **"Start Session"**（紫色大按钮）
3. ✅ 等待 1-2 秒（看到欢迎消息）
4. ✅ 在右侧**底部**看到输入框
5. ✅ 输入 "hello"
6. ✅ 按回车或点发送按钮
7. ✅ 看到 AI 回复

## ❓ 常见问题

### Q: 为什么一开始看不到输入框？

**A:** 这是设计如此！输入框只在**建立连接后**才显示。

**原因：**
- 防止在未连接时发送消息
- 更好的用户体验
- 清晰的状态指示

---

### Q: 点击 Start Session 后什么都没发生？

**检查：**

1. **后端是否运行？**
```bash
curl http://localhost:8000/api/health
```

应该返回 JSON，包含 `"status":"healthy"`

2. **前端是否运行？**
```bash
curl http://localhost:3000
```

应该返回 HTML

3. **浏览器控制台有错误吗？**
- 按 F12 打开开发者工具
- 查看 Console 标签
- 查看是否有红色错误

---

### Q: 输入框是灰色的，不能输入？

**原因：** 正在处理中或连接断开

**解决：**
1. 等待当前消息处理完成
2. 或者点击"挂断"按钮，重新 Start Session

## 💡 专业提示

### 快捷键：
- `Enter` - 发送消息
- `Shift + Enter` - 换行（如果需要多行）

### 聊天记录：
- 用户消息：紫色气泡，靠右
- AI 消息：灰色气泡，靠左

- 连接中：显示欢迎消息
